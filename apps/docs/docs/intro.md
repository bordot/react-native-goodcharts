# react-native-goodcharts

`react-native-goodcharts` is a monorepo-built charting library for React Native focused on Skia rendering, composable APIs, and accessible defaults.

## Packages

- `packages/charts`: the publishable library
- `apps/example`: Expo Router gallery app
- `apps/docs`: Docusaurus documentation site

## Current Surface

The scaffold exports the P0 chart surface, themes, composable primitives, hooks, and benchmark/test scaffolding required to build toward `v1.0`.
