# Cartesian Gallery

The current scaffold ships the following cartesian chart components:

- `LineChart`
- `AreaChart`
- `BarChart`
- `HorizontalBarChart`
- `StackedBarChart`
- `ScatterChart`
- `ComboChart`
- `CandlestickChart`

All of them reuse the same chart state pipeline, axes, legends, tooltip plumbing, accessibility label generation, and selection/crosshair overlays.
